# Cafe Order System

A web application designed to simplify the ordering workflow for small cafes.

The system helps cafe staff manage suppliers, organize frequently ordered items, track previous orders, and generate supplier order messages quickly. This project was built to solve a real-world problem where cafe owners repeatedly typed similar order requests every week.

## Features

### Vendor Management

* Create and manage supplier information
* Store contact details for each vendor
* Organize vendors in a centralized dashboard

### Item Management

* Create and manage orderable items
* Associate items with vendors
* Store reusable ordering information

### Order History

* Record previous orders
* Review historical order data
* Track ordering activity over time

### Message Generation

* Generate supplier order messages with a single click
* Reduce repetitive manual typing
* Standardize communication with suppliers

### Template Management

* Create reusable order templates
* Quickly generate recurring orders
* Improve ordering efficiency

## Problem

Many small cafes place orders through messaging applications such as SMS, KakaoTalk, or Messenger.

The cafe owner this system was built for regularly needed to:

1. Check which items to order
2. Review previous orders
3. Manually write supplier messages
4. Repeat the same process every week

This repetitive workflow was time-consuming and prone to mistakes.

---

## Solution

The Cafe Order System centralizes supplier and item management while providing one-click message generation for suppliers.

Instead of manually writing similar messages every week, staff can generate structured order requests directly from the application.
